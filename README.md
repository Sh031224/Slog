# Slog

📖 My Blog that contains various articles such as my activities and thoughts.

## 기술 스택

### Front-End

- React & SSR
- sass
- FCM

### Back-End

- Express
- TypeORM
- MySQL
- FCM

## 기능

### 게시글

- 작성, 임시저장
- 조회수
- 댓글 및 답글

> FCM을 이용하여 댓글이 생성되면 게시글 작성자에게 알림, 답글이 생성되면 댓글 작성자에게 알림이 가도록 설정하였습니다.

### 로그인

- Facebook 연동
